# Banking API Challenge
### Set up and run the app
---

1. Build api dependencies.
```
// yarn
$ yarn

// npm
$ npm install
```
2. Copy `.env.sample` and rename to `.env`
3. Modify `DATABASE_URL` for the mongodb connection.
4. Start the app.
```
// yarn
$ yarn start

// npm
$ npm start
```
